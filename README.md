## APIs

### Sherwin-Williams Supplier API

The Sherwin-Williams Supplier API enables B2B partners to integrate supply chain workflows, manage purchase orders, submit invoices, and exchange product and inventory data with Sherwin-Williams enterprise systems. The portal supports EDI and API-based integrations for suppliers and trading partners.

**Human URL:** [https://suppliers.sherwin-williams.com/](https://suppliers.sherwin-williams.com/)

### Sherwin-Williams EDI Integration

Sherwin-Williams EDI integration capabilities enable trading partners to exchange electronic data interchange (EDI) documents including purchase orders, invoices, advance ship notices, and inventory feeds. Supported EDI formats include X12 and EDIFACT standards.

**Human URL:** [https://suppliers.sherwin-williams.com/](https://suppliers.sherwin-williams.com/)
